
# محاكاة نظام أبشر — بصمة سلوكية (React CDN demo)

هذه نسخة جاهزة للرفع على GitHub Pages أو استضافة ثابتة. تستخدم React عبر CDN وChart.js لعرض الرسوم.
تحتوي على:
- لوجو SVG رسمي مصمم خصيصًا.
- لوحة تحكم متطورة مع أزرار إنشاء والتحقق.
- رسوم بيانية تعرض حركة الماوس وفواصل الضغطة.
- محاكاة لالتقاط حركة المستخدم وإنتاج بصمة سلوكية (متجه سمات).
- API وهمي (MockAPI) مدمج في الواجهة يعيد وعودًا مشابهة لنداءات حقيقية.

لتشغيل محليًا: فقط افتح `index.html` في المتصفح أو قم برفع المشروع إلى أي استضافة ثابتة.
